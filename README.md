## 📖 使用指南

### 1️⃣ 填寫前置資料

開啟工具後，首先填寫施工前置資料：

- **施工日期**：自動帶入當天日期，可調整
- **填表人員**：從清單選擇施工人員
- **班別**：選擇早班/晚班/加班/支援/臨時/其他
  - 💡 早班顯示白色背景，晚班自動切換深色模式
- **狀態**：已完工/未完工
- **業主**：選擇施工業主
- **施工地點**：從區域分類清單選擇
- **車輛**：選擇使用車輛

填寫完成後點擊「**開始統計**」進入主程式。

### 2️⃣ 選擇標線類型

在主程式頁面：

1. 從左側清單選擇要統計的標線類型
2. 標線依據寬度和用途分為 A-H 八大類：
   - **A 區**：10cm 道路標線（白線、黃線、雙黃線等）
   - **B 區**：15cm 道路標線（白邊線、待轉區等）
   - **C 區**：40cm 道路標線（停止線、行穿線等）
   - **D 區**：停等區與車格類
   - **E 區**：字模標線（箭頭、符號、文字等）
   - **F 區**：內外型標線（黃網線、槽化線等）
   - **G 區**：其他與自訂項目
   - **H 區**：非標線項目（貓眼、防撞桿等）

### 3️⃣ 輸入數據

1. 使用觸控螢幕或鍵盤輸入數字
2. 畫面上方會即時顯示計算公式與結果
3. 確認無誤後點擊「**確認**」或按 **Enter** 鍵

範例：
```
選擇：白線
輸入：10.5 (長度 10.5 公尺)
計算：10.5m × 0.1 = 1.05m²
```

### 4️⃣ 檢視統計列表

右側顯示所有已輸入的項目：
- 標線名稱與輸入值
- 計算係數與面積
- 總計面積（頂部顯示）
- 可點擊垃圾桶圖示刪除錯誤項目

### 5️⃣ 複製報表

完成統計後：

1. 點擊「**複製報表 (P)**」按鈕或按鍵盤 **P** 鍵
2. 報表已複製到剪貼簿
3. 貼到 Line、Excel 或任何文字編輯器

報表範例：
```
【道路標線施工日報表】
日期: 2026-01-12
人員: 陳易璘 | 班別: 早班 | 狀態: 已完工
業主: 磐碩8上 | 車輛: A車
地點: 板橋區
------------------------
白線:(3+5+2)=10m*0.1=1.00m²
停止線:15m*0.4=6.00m²
1m行穿線:8條*1*0.4=3.20m²

標線總計:(1.00+6.00+3.20)=10.20m²
------------------------
【非標線施工項目】
黃貓眼總計:(5+3)=8個
```

## 🎨 自訂項目

如果預設的標線類型不符合需求，可以新增自訂項目：

1. 滾動到標線類型清單底部
2. 點擊「**新增自訂項目**」
3. 填寫資訊：
   - **項目名稱**：例如「特殊圖案」
   - **計算方式**：長度(m)、數量(個/點/條)、計次(無面積)
   - **係數算式**：例如 `0.15` 或 `1.0*0.5`
4. 預覽公式確認無誤後點擊「**新增**」

自訂項目會自動儲存，下次開啟時仍然存在。

## 🔧 進階功能

### 修改規格

如果某個標線類型的係數需要調整：

1. 選擇該標線類型
2. 點擊輸入畫面上方的「**修改**」按鈕
3. 輸入新的係數算式（例如從 `0.1` 改為 `0.12`）
4. 確認後，該類型的計算方式即時更新

### 恢復預設

如果自訂項目太多或係數設定錯誤：

1. 點擊標線類型清單上方的「**恢復預設**」
2. 確認後會移除所有自訂項目，恢復原始設定

⚠️ **警告**：此操作會刪除所有自訂項目，請謹慎使用。

### 聲音控制

右上角有聲音開關按鈕：
- 🔊 **開啟**：語音播報與音效提示
- 🔇 **關閉**：靜音模式

## 💾 資料管理

### 自動儲存

所有資料自動儲存在瀏覽器的 LocalStorage：
- 自訂項目
- 標線類型修改

### 資料清除

如需清除資料：

1. **清空統計列表**：點擊「清空」按鈕（只清除當前統計）
2. **清除瀏覽器資料**：
   - Chrome: 設定 → 隱私權和安全性 → 清除瀏覽資料 → 選擇「Cookie 和其他網站資料」
   - Edge: 設定 → 隱私權、搜尋與服務 → 清除瀏覽資料

## ❓ 常見問題

### Q: 資料會不會遺失？

A: 資料儲存在瀏覽器本地，除非清除瀏覽器資料或更換裝置，否則不會遺失。建議定期複製報表備份。

### Q: 可以多人共用嗎？

A: 目前版本僅支援單機使用，每個瀏覽器的資料獨立。如需團隊共用，建議複製報表到共用文件。

### Q: 如何新增人員或業主？

A: 目前需要修改 HTML 檔案中的 `FORM_OPTIONS` 物件。未來版本將提供配置檔案功能。

### Q: 計算結果不正確怎麼辦？

A: 請檢查：
1. 輸入的數值是否正確
2. 選擇的標線類型是否正確
3. 係數是否正確（可用「修改」功能調整）

### Q: 語音不會播報？

A: 請確認：
1. 聲音開關是否開啟
2. 瀏覽器是否支援語音合成（Safari 部分版本不支援中文）
3. 系統音量是否開啟

### Q: 可以匯出 Excel 檔案嗎？

A: 目前支援文字格式報表，複製後可貼到 Excel 並手動格式化。未來版本將考慮加入直接匯出功能。

# trafficline (道路標線施工統計工具)

這是一個專為道路標線施工現場設計的數據統計與報表管理工具。透過簡約高效的網頁介面，施工人員可以在現場即時記錄各類標線的施作面積、長度與數量，並自動生成格式化報表。

## 🚀 核心功能

- **即時統計**：分類管理 10cm/15cm/40cm 標線、字模、內外型標線等。
- **自訂項**：支援自訂行穿線長度與特殊的字模面積。
- **智慧操作**：支援鍵盤快捷鍵 (WASD + Enter) 與語音音效回饋。
- **拖曳管理**：透過拖曳輕鬆調整統計項目的順序。
- **自動報表**：一鍵複製 LINE 或 Excel 用途的純文字統計表，或下載 PDF 電子版報表。
- **前置作業**：整合人員、業主、車輛與地點的施工前置資訊管理。

## 📦 技術規格

- **前端框架**：Vanilla JavaScript + Tailwind CSS
- **圖示庫**：Lucide Icons
- **報表工具**：html2canvas / html2pdf.js / XLSX.js
- **部署方式**：單一 HTML 檔案，支援離線執行。

## 🛠 安裝與使用

1. 下載或 Clone 本專案。
2. 開啟 `index.html` 即可在瀏覽器直接運行。
3. 填寫前置作業後，即可點選各類項目進行統計。

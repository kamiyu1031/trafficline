# 使用範例

本文件提供道路標線施工統計工具的實際使用範例。

---

## 範例一：基本標線施工記錄

### 場景
板橋區白線與停止線施工，早班作業。

### 操作步驟

#### 1. 填寫前置資料
```
施工日期：2026-01-12
填表人員：陳易璘
班別：早班  ← 選擇後背景變為白色
狀態：已完工
業主：磐碩8上
施工地點：板橋區
車輛：A車
```

點擊「開始統計」。

#### 2. 輸入施工數據

**白線 3 段：**
- 選擇「白線」
- 輸入 `15`
- 按 Enter 確認 → 15m × 0.1 = 1.5m²
- 輸入 `20`
- 按 Enter 確認 → 20m × 0.1 = 2.0m²
- 輸入 `8.5`
- 按 Enter 確認 → 8.5m × 0.1 = 0.85m²

**停止線：**
- 選擇「停止線」
- 輸入 `12`
- 按 Enter 確認 → 12m × 0.4 = 4.8m²

#### 3. 複製報表

按鍵盤 `P` 或點擊「複製報表」，得到：

```
【道路標線施工日報表】
日期: 2026-01-12
人員: 陳易璘 | 班別: 早班 | 狀態: 已完工
業主: 磐碩8上 | 車輛: A車
地點: 板橋區
------------------------
白線:(15+20+8.5)=43.5m*0.1=4.35m²
停止線:12m*0.4=4.8m²

標線總計:(4.35+4.8)=9.15m²
```

### 場景
新莊區夜間黃線施工。

### 操作步驟

車輛：B車

**效果：** 畫面自動切換為深色主題，減少夜間眩光。

#### 2. 輸入施工數據

**雙黃線：**
- 選擇「雙黃線」
- 輸入 `50`
- 按 Enter 確認 → 50m × 0.1 × 2.0 = 10.0m²

## 範例三：行穿線施工

### 場景
大安區行人穿越線施工。

### 操作步驟

#### 輸入2m行穿線（8條）

- 選擇「2m行穿線」
- 輸入 `8`（代表 8 條）
- 按 Enter 確認 → 8條 × 2.0 × 0.4 = 6.4m²

#### 報表格式

```
2m行穿線:8條*2*0.4=6.4m²
```

## 範例四：自訂項目

### 場景
施工中遇到特殊圖案標線，需要自訂項目。

### 操作步驟

#### 1. 新增自訂項目

- 滾動到標線類型底部
- 點擊「新增自訂項目」
- 填寫：
  ```
  項目名稱：特殊交通島圖案
  計算方式：數量 (個/點/條)
  係數算式：2.5*0.3
  ```
- 預覽：`1點 × 2.5*0.3 = 0.75 m²`
- 點擊「新增」

#### 2. 使用自訂項目

- 選擇「特殊交通島圖案」
- 輸入 `6`（6 組圖案）
- 按 Enter 確認 → 6 × 0.75 = 4.5m²

## 範例五：複雜施工項目組合

### 場景
中正區綜合標線施工。

### 施工清單
✅ 白邊線 120m  
✅ 停止線 2 條（各 6m）  
✅ 1m行穿線 10 條  
✅ 指標箭頭 5 個  
✅ 黃貓眼 15 個（非標線項目）

### 輸入順序

1. **白邊線** → 輸入 `120` → 120m × 0.15 = 18.0m²
2. **停止線** → 輸入 `6` → 6m × 0.4 = 2.4m²
3. **停止線** → 輸入 `6` → 6m × 0.4 = 2.4m²
4. **1m行穿線** → 輸入 `10` → 10條 × 1 × 0.4 = 4.0m²
5. **指標箭頭** → 輸入 `5` → 5 × 1 = 5.0m²
6. **黃貓眼** → 輸入 `15` → 15個（數量統計）

### 最終報表

```
【道路標線施工日報表】
日期: 2026-01-12
人員: 許朝淵 | 班別: 早班 | 狀態: 已完工
業主: 磐碩 | 車輛: C車
地點: 中正區
------------------------
白邊線:120m*0.15=18.00m²
停止線:(6+6)=12m*0.4=4.80m²
1m行穿線:10條*1*0.4=4.00m²
指標箭頭:5個*1=5.00m²

標線總計:(18.00+4.80+4.00+5.00)=31.80m²
------------------------
【非標線施工項目】
黃貓眼總計:15個
```

## 範例六：鍵盤快速操作

### 場景
熟練使用者完全使用鍵盤操作。

### 完整鍵盤流程

#### 前置頁面
```
1. 打開工具
2. Tab 鍵移動到日期欄位，修改日期（如需）
3. W/S 鍵上下移動，A/D 鍵左右移動
4. 空白鍵選擇「陳易璘」
5. S 鍵向下，空白鍵選擇「早班」
6. S 鍵向下，空白鍵選擇「已完工」
7. S 鍵向下，空白鍵選擇業主
8. S 鍵向下，D 鍵移動，空白鍵選擇地點
9. S 鍵向下，空白鍵選擇車輛
10. Enter 鍵進入主程式
```

#### 主程式頁面
```
1. W/S/A/D 鍵移動到標線類型
2. 空白鍵選擇標線類型
3. 直接按數字鍵 1 0 → 輸入 10
4. Enter 確認
5. 重複步驟 1-4 繼續輸入
6. 完成後按 P 鍵複製報表
```

⏱️ **熟練後可在 30 秒內完成一整份報表！**

## 範例八：修改規格係數

### 場景
白線實際寬度為 12cm（而非預設 10cm）。

### 操作步驟

#### 方法一：單次修改
1. 選擇「白線」
2. 點擊畫面上方「修改」按鈕
3. 輸入新係數：`0.12`
4. 確認
5. 之後所有白線都用 0.12 計算

#### 方法二：臨時自訂項目
1. 點擊「新增自訂項目」
2. 名稱：`12cm白線`
3. 計算方式：長度(m)
4. 係數：`0.12`
5. 新增後選擇使用

## 範例九：錯誤修正

### 場景
輸入錯誤需要修正。

### 錯誤類型與解決方式

#### 1. 輸入數字錯誤（尚未確認）
- 按 `Backspace` 或點擊 `DEL` 逐位刪除
- 或按 `Escape` 或點擊 `AC` 全部清空

#### 2. 已確認但錯誤
- 在右側列表找到該項目
- 點擊垃圾桶圖示刪除
- 重新輸入正確數值

#### 3. 選錯標線類型
- 點擊刪除
- 重新選擇正確類型並輸入

## 範例十：報表技巧

### 貼到 Line

複製報表後，直接貼到 Line 對話框：

```
【道路標線施工日報表】
日期: 2026-01-12
...
```

格式完整保留，清晰易讀。

### 貼到 Excel

1. 複製報表
2. 開啟 Excel
3. 貼上到儲存格 A1
4. 資料會以純文字格式貼上
5. 可進一步格式化或計算

### 儲存到文字檔

```
1. 複製報表
2. 開啟記事本
3. 貼上
4. 另存為：施工日報_2026-01-12.txt
```

## 常見問題處理

### Q: 輸入小數時無法再次輸入小數點？
A: 每個數字只能有一個小數點，這是正常防呆設計。

### Q: 為何 Enter 沒反應？
A: 確認已輸入數字，且數字大於 0。

### Q: 語音不會播報？
A: 檢查右上角聲音圖示是否為靜音狀態。

### Q: 報表複製後格式跑掉？
A: 建議使用「保留格式」貼上，或貼到純文字編輯器。

## 效率提升技巧

### 🚀 快速技巧

1. **記住常用標線的分類位置**
   - A區：基本線型
   - D區：車格
   - E區：符號文字

2. **善用鍵盤導航**
   - 純鍵盤操作速度最快
   - 熟記 WASD + 空白 + Enter

3. **排序輸入**
   - 先輸入同類型標線
   - 減少選擇切換次數

4. **即時檢查**
   - 注意右側列表總計
   - 隨時檢查計算是否正確
